# i2c_lcd_display
ESP32 LCD I2C project using ESP-IDF demonstrates interfacing a 16x2 (0802A) LCD with ESP32 via I2C. It initializes I2C master, configures LCD, clears screen, and displays text on both lines. Built with FreeRTOS, it shows basic embedded programming, driver usage, and peripheral communication, useful for IoT and learning ESP32 development.
